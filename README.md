# Football-Analytics-Project-Bournemouth-vs-Manchester-United

# Overview

This project analyzes a football match between Manchester United and Bournemouth using shot data and Expected Goals (xG) to evaluate attacking performance, chance quality, and efficiency.

The goal is to demonstrate how data analytics can be applied in football to uncover insights about team tactics and player performance.

# Key Objectives
Analyze shot distribution and locations
Evaluate chance quality using xG
Compare attacking efficiency between teams
Break down player-level performance
Visualize insights using Python
# Tools & Technologies

Matplotlib

Data scraping (SofaScore API)

📈 Key Findings
# Attacking Performance
Manchester United generated higher xG (~1.77) compared to Bournemouth (~1.56)
Both teams concentrated their attacks inside the penalty area
Limited long-range shots indicate disciplined shot selection
# Shot Outcomes
Manchester United:
High number of blocked shots
Struggled to break defensive structure
Bournemouth:
Higher number of missed chances
One attempt hit the post (unlucky moment)

# Insight:
Efficiency in finishing played a key role, as both teams underperformed relative to their chances.

# Player Analysis
Bruno Fernandes
Highest shot involvement
Mixed outcomes (goal, saves, blocks)
Key attacking contributor
Ryan Christie & Eli Junior Kroupi
Efficient finishing
Converted chances effectively

# Body Part Analysis
Right foot: Most dominant (majority of shots)
Left foot: Moderate contribution
Headers: Very limited
Indicates low aerial threat and emphasis on ground play.

# Shot Map Insights
Shots were clustered inside the box
One high-quality chance (high xG)
Majority of attempts were low-to-moderate xG
Suggests limited creation of clear-cut chances

# Conclusion

The match was closely contested in terms of chance quality, with Manchester United holding a slight edge in xG. However, neither team demonstrated clinical finishing.

Manchester United → struggled with shot execution (blocked attempts)
Bournemouth → struggled with finishing accuracy (misses)

# This highlights an important concept in football analytics:

Creating chances is important, but converting them is what wins matches.
